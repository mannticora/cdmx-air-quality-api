🌆 CDMX Air Quality API

Guía interactiva — datos reales de calidad del aire de la Ciudad de México

Esta API almacena y sirve datos meteorológicos reales de las estaciones de monitoreo de la Red REDMET de la CDMX. Contiene más de 33,000 mediciones reales de 2026.
1
¿Qué datos tiene?
Temperatura (TMP), humedad relativa (RH), velocidad del viento (WSP) y dirección del viento (WDR) — registradas cada hora por 26 estaciones de monitoreo.
2
¿De dónde vienen los datos?
Del sistema oficial AIRE CDMX (aire.cdmx.gob.mx), la red oficial de monitoreo atmosférico del gobierno de la Ciudad de México.
3
¿Cómo se consultan?
Enviando peticiones HTTP a los endpoints. Esta guía te muestra cómo hacerlo de forma interactiva — sin escribir código.
4
¿Qué zonas cubre?
Norte, Sur, Centro, Oriente, Poniente y Noreste — las zonas de monitoreo metropolitano de la CDMX.

Ejemplo con JavaScript (fetch)

// Obtener temperatura de la zona Norte fetch('https://cdmx-air-quality-api-production.up.railway.app/measurements/?zone=Norte&pollutant=tmp&limit=10') .then(res => res.json()) .then(data => { console.log(data); // Array de mediciones });

Ejemplo con Python (requests)

import requests url = "https://cdmx-air-quality-api-production.up.railway.app/measurements/" params = { "zone": "Sur", "pollutant": "tmp", "start": "2026-01-01T00:00:00", "limit": 50 } response = requests.get(url, params=params) data = response.json() print(data)

Ejemplo con React

import { useEffect, useState } from "react"; const API = "https://cdmx-air-quality-api-production.up.railway.app"; function AirQualityWidget() { const [data, setData] = useState([]); useEffect(() => { fetch(`${API}/measurements/?pollutant=tmp&limit=5`) .then(r => r.json()) .then(setData); }, []); return ( <ul> {data.map(m => ( <li key={m.id}>{m.station}: {m.value}°C</li> ))} </ul> ); }

Crear una medición (POST)

fetch('https://cdmx-air-quality-api-production.up.railway.app/measurements/', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ station: "PED", zone: "Sur", pollutant: "tmp", value: 22.5, unit: "°C", timestamp: "2026-01-15T14:00:00" }) }).then(r => r.json()).then(console.log);
